# Door-Cutin 使用指南

****一个为 Foundry VTT 场景互动打造的沉浸式插件：**
 开门、陷阱触发瞬间产生特写 Cut-in 动画/图像，提升玩家的情绪张力与地下城探索感。**

## 插件简介

****Door-Cutin 最初用于在打开门时触发特写画面，逐渐扩展到陷阱触发、机关交互等事件。**
 通过将画面与音效结合，DM 可以轻松设计更复杂的场景互动，打造极具沉浸感的地下城体验。**

**适合玩法：**

- 地下城探索（Dungeon Crawl）
- Rogue-like 地图
- Puzzle / 机关解谜场景
- Boss 入口演出
- 剧情关键事件

------

## 核心功能

- 门互动触发 Cut-in
- 陷阱触发 Cut-in
- 支持静态图像与 WebM 动画
- 支持搭配 Sequencer 进行音效/特效播放
- 可绑定到 Tile / Door 等实体

------

## 依赖

本插件依赖：

- **Sequencer**

------

## 使用方法（快速入门）

1. 在世界中创建 门 / Tile / 陷阱元素
2. 进入对应的配置窗口
3. 在 Door-Cutin 区块中启用并配置：
   - 触发资源（图片 or webm）
   - 显示位置 / 缩放
   - 与 Sequencer 的协同效果（可选）

完成后，玩家触发事件即可看到演出 Cut-in。

------

## 设计原则

- DM 无需复杂脚本即可制作演出效果
- 保持资源与逻辑解耦
- 支持拓展：陷阱、机关、剧情触发均可用
